# Zomato Data Analysis & Rating Prediction

## 📌 Overview
This project analyzes restaurant data from Bangalore to identify factors influencing customer ratings.

## 🛠️ Tools Used
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## 📊 Key Insights
- Online ordering improves ratings
- Cost does not strongly affect ratings
- Popular restaurants have better ratings

## 🤖 Machine Learning
Built a regression model to predict restaurant ratings.
